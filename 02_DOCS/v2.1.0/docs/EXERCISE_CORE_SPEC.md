# Exercise Core Spec v2.1.0

## Zweck

Diese Spezifikation beschreibt die neuen Übungsdaten.

## Wichtig

Die Daten dürfen groß sein. Die UI im Training bleibt klein.

## Quick Training Mode

Im Training sichtbar:

- Übungsname
- letzte Leistung
- heutige Sätze
- kompakter LWS-Hinweis
- Alternative-Button nur bei Bedarf

Nicht sichtbar:

- Technikhinweis
- häufige Fehler
- lange Beschreibungen
- komplette Muskelkarte
- vollständige Historie

Diese Informationen gehören in den Übungsdetailbereich.
