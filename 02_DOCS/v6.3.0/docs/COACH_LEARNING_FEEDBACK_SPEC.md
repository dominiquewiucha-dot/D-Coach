# Coach Learning & Feedback

Der Nutzer kann Empfehlungen bewerten.

Das System lernt lokal:
- welche Empfehlungen hilfreich sind,
- ob Vorschläge zu aggressiv oder zu vorsichtig sind,
- welche Übungen bevorzugt werden.

Sicherheits- und LWS-Regeln bleiben unveränderlich.
