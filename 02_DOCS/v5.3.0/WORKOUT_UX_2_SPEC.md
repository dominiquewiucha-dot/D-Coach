# Workout UX 2.0

## Day-selection screen

Top:
- page title,
- active plan,
- selected day,
- duration,
- dominant Start button.

Below:
- compact Smart Builder preview.
- no duplicated full workout list unless expanded.

## Active workout

- sticky compact progress header,
- one exercise in focus,
- large weight/repetition controls,
- last performance visible,
- optional RIR behind a compact expansion,
- `Gerät besetzt?` alternative action,
- rest timer integrated without a modal.

## Information density

The user must be able to start the session without reading paragraphs.
