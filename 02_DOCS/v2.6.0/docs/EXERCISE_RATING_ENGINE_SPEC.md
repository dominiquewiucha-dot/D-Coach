# Exercise Rating Engine Spec v2.6.0

## Zweck
D-Coach bewertet Übungen nach Qualität für den konkreten Nutzer.

## Scores 0–10
- Muskelreiz
- Ermüdungskosten
- Zeiteffizienz
- Progressionspotenzial
- Technikanspruch
- Stabilitätsanforderung
- LWS-Freundlichkeit
- Schulter-Freundlichkeit
- Knie-Freundlichkeit
- Wiedereinstieg
- Muskelaufbau
- Kraftaufbau

## UI
Ratings gehören in Übungsdetails, Alternativen und Coach Mode. Nicht in den Quick Training Satzscreen.
