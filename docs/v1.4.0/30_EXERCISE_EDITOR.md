# Exercise Editor v1.4.0

## Ziel

Nutzer kann eigene Übungen anlegen und bearbeiten.

## Felder

- Name
- Equipment
- Zielmuskel
- Hilfsmuskeln optional
- LWS-Eignung
- Sätze
- Wiederholungen
- Pause
- Notiz
- Alternative zu anderer Übung optional

## Schutzregel

Wenn eine Übung Historie hat:

- nicht hart löschen
- nur archivieren

## Eigene IDs

Custom IDs:

```text
custom_ex_...
```
