# Accessibility und Mobile UX v1.4.0

## Ziel

D-Coach soll im Gym schnell und mit einer Hand bedienbar sein.

## Regeln

- Touchflächen mindestens 44 px
- wichtige Buttons unten erreichbar
- Gewicht/Reps schnell editierbar
- hoher Kontrast
- keine winzigen Texte
- keine Pflicht zum Scrollen während Satz-Eingabe, wenn möglich

## Training Screen

Priorität:

1. Übung
2. letzte Leistung
3. heutige Satzfelder
4. nächster Satz / nächste Übung
