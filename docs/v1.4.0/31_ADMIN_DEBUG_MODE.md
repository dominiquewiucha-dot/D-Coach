# Admin und Debug Mode v1.4.0

## Ziel

Fehler schneller finden.

## Funktionen

- App-Version anzeigen
- geladene Datenbanken anzeigen
- Storage-Größe anzeigen
- Backup herunterladen
- Testdaten laden
- Cache leeren
- Service Worker Status
- letzte Fehler anzeigen

## Aktivierung

In Einstellungen:

```text
Debug-Modus aktivieren
```

oder versteckt:

```text
5x auf Versionsnummer tippen
```
