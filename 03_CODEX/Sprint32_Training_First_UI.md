# Sprint 32 – Training First UI

## Ziel

Training-Screen vereinfachen und Informationsflut vermeiden.

## Aufgaben

1. Training-Screen prüfen.
2. Alles entfernen oder einklappen, was nicht für aktuellen Satz nötig ist.
3. Letzte Leistung prominent anzeigen.
4. Satzfelder groß und schnell bedienbar machen.
5. Timer kompakt anzeigen.
6. Alternative-Button nur sekundär.
7. LWS-Hinweis kompakt.
8. Charts/Statistiken aus Training-Screen entfernen.

## Abnahmekriterien

- Satz kann in wenigen Sekunden eingetragen werden.
- keine großen Charts im Training.
- keine langen Texte im Training.
- eine Hand Bedienung möglich.
- bestehende Funktionen bleiben erreichbar.
