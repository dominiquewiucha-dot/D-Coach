# D-Coach PWA

Offline-fähige Trainings-App für iPhone/Browser.

## Nutzung

Diese PWA ist statisch und kann direkt über GitHub Pages, Netlify oder Vercel gehostet werden.

## Lokal starten

```powershell
powershell -ExecutionPolicy Bypass -File .\serve.ps1
```

Dann öffnen:

```text
http://127.0.0.1:5177
```

## iPhone

Nach dem Deployment die HTTPS-URL in Safari öffnen:

1. Teilen
2. Zum Home-Bildschirm
3. Öffnen wie eine App

## Daten

Trainings- und Gewichtsdaten werden lokal im Browser gespeichert.
